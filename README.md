# ConnectFourAgent
C++ program to play Connect Four against an algorithmic agent

# How to run locally
Compile with: <br />
1.```g++ Game.cpp -o Game.o``` <br />
2. ```g++ Heuristic.cpp -o Heuristic.o``` <br />
3. ```g++ main.cpp -o main.o``` <br />
4. ```g++ Game.o Heuristic.o main.o -o ConnectFour```

Play with: <br />
```./ConnectFour```
